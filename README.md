# Vunor

A Vue 3 UI component library built on UnoCSS. Ships a custom UnoCSS preset, a design-token-driven palette engine, golden-ratio typography and spacing, and 30+ ready-to-use components powered by Reka UI.

Supports **Vue 3 + Vite** and **Nuxt 3**.

See [`packages/vunor/`](packages/vunor/) for full documentation.
